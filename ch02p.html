HTML 소개와 개념

HTML이란 무엇인가요?

HTML(Hyper Text Markup Language)은 웹 페이지를 만드는데 사용되는 마크업 언어입니다. HTML은 웹 페이지의 구조와 콘첸츠를 정의하는 역할을 합니다. 이것은 웹 브라우저에게 어떻게 페이지를 표시해야 하는지 알려주는데 사용합니다.

HTML의 주요 특징

  • "마크업 언어" HTML은 마크업 언어로, 텍스트를 태그로 둘러써서 구조를 정의합니다.
  • "계층 구조" HTML 문서는 계층 구조를 가지며 요소들을 주업할 수 있습니다.
  • "표준화" HTML은 우베 표준으로 관리되며 W3C(World Wide Web Consortium)에서 관리합니다.
  • "크로스 플랫폼" HTML은 모든 주요 웹 브라우저에서 지원되며, 모바일기기와 호환됩니다.

HTML 기본 구조

HTML문서는 보통 head, title, body등의 요소로 시작합니다. html 요소는 문서의 루트 요소이며, head 요소에는 문서의 메타 정보와 제목이 들어갑니다. body 요소에는 실제 콘텐츠가 들어갑니다.
